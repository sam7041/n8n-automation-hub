## 🛠️ How to Use
1. **Host n8n**: Ensure you have a self-hosted or cloud instance of n8n.
2. **Import**: Download a `.json` template from this repo and select **"Import from File"** in your n8n canvas.
3. **Configure**: Enter your own API credentials (OpenAI, Google, etc.) in the respective nodes.
